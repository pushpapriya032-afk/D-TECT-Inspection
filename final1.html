🔍 Weapon Detection Prototype

What this demo does

Inspective Detective System combines live camera input with a simple detection simulation (replaceable with ML model). On detection it will attempt to send an SMS alert (Textbelt demo or API) and visually signal an alert.

Quick Start

Start camera and test detection from your browser (localhost/HTTPS required).

Alerting

Uses Textbelt or native SMS fallback to notify contacts.

Demo Mode

Random simulated detections for safe testing.

Extendable

Swap detectWeapon() with an ML model or server API.
